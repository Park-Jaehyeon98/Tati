## 주요기능

### 1) 로그인

![스크린샷 2023-08-17 오후 11.54.34.png](/img/7.png)

- 회원 가입한 이메일과 비밀번호를 입력해 로그인
- 로그인을 완료하면, 플랫폼 메인페이지로 이동

### 3) 스터디 관련 기능

![Untitled](/img/10.png)

- 스터디룸 입장(캠스터디) - 입실 시간 10분전에 입장 버튼 활성화
- 가입 신청 현황에서 방장이 신청한 회원을 선별하여 승인, 거절 기능
